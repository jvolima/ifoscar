# Ifoscar

## Funcionalidades
### ADM
- Cadastrar alunos a partir de um arquivo csv ou por um formulário
- Cadastrar video clipes
- Atualizar video clipes
### Alunos
- Visualizar lista de video clipes
- Votar em um video clipe
